Quality control check of CTD data files and conversion to ASCII MedAtlas format. Figures and metadata (CDIs) are also generated.

  • 1 Select files
  • 2 Check report

After downloading, you will get a zip file containing:

  • Data (depending on your initial choice):
    • File in ASCII MedAtlas format containing all the stations.
    • Flagged (quality control) CNV files adapted to be easilly ingested in NEMO software.
  • Figures: some basic plots of temperature and salinity profiles. Bad flags are marked over the plot. Shaded regions indicate climate values obtained from World Ocean Atlas.
  • Metadata: XML (one file per cast) following SeaDataNet requirements (also known as CDIs)
